== Description ==
Unlock the power of Zoho Integration for WordPress, your ultimate solution for capturing leads and streamlining user registration seamlessly. This advanced plugin connects your WordPress website with Zoho CRM, ensuring effortless lead generation and contact management.


== Key Features: ==

* **Effortless Integration:** Say goodbye to manual work and coding. Zoho Integration for WordPress simplifies the integration process.
* **User-Friendly Interface:** Our intuitive interface makes setup a breeze, no HTML knowledge required.
* **Endless Customization:** Create, add, and edit fields at any time to tailor your forms to your exact needs.
* **Dynamic Form Generation:** Generate web forms on the fly, capturing your website's visitors as high-potential leads in Zoho CRM.
* **Shortcode Integration:** Seamlessly embed forms with a shortcode, enhancing user engagement.
* **Fetch Zoho CRM Fields:** Fetch CRM Zoho lead fields directly into WordPress to create a customized form.
* **Field Customization:** Enable/disable fields to include in the form or not, ensuring complete control.
* **Widget Integration:** Use a separate shortcode to integrate forms as widgets site-wide, expanding your reach.
* **Sync WP Members:** Capture and sync newly registered WP members to Zoho CRM contacts, ensuring a centralized database for your leads.
* **Part of WP Leads Builder for Any CRM:** Zoho Integration is now part of the powerful [WP Leads Builder For Any CRM](https://wordpress.org/plugins/wp-leads-builder-any-crm), offering even more features and CRM integrations.

== Installation ==

1. First, install [CRM WordPress Leads](https://wordpress.org/plugins/wp-leads-builder-any-crm)
2. Then Extract the wp-zoho-crm.zip in wordpress/wp-content/plugins using FTP or through plugin install in wp-admin.
3. Activate the plugin in the WordPress plugin list.

== Frequently Asked Questions ==

1) How to create form with Contact Form 7?<br />

* Enable the Contact Form 7 in CRM configuration under Form Type.
* In CRM Forms menu select the required fields and Save.
* The shortcode with selected fields created in Contact Form 7.
* Place the shortcode in Post, Page or Widget.

2) Why I'm not able to submit the form created with Contact form?<br />

* After generating the shortcode for Contact form.
* Go to Contact â†’ Contact Forms â†’ Edit view of the shortcode.
* In Mail menu, configure all the settings and save.
* If the mail is not configured the form will trigger an error message stating 'Submission failed'.
